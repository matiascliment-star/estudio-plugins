# Checklist de Control de Pericia Medica Laboral

Metodologia del Estudio Garcia Climent. Cada control se evalua como: **OK**, **WARNING** (advertencia) o **ERROR** (motivo de impugnacion).

---

## SECCION 1: Datos Basicos del Accidente

### 1.1 Fecha del accidente

**Que verificar:** La fecha del accidente debe coincidir en demanda, pericia y SRT (si hay).

| Resultado | Condicion |
|-----------|-----------|
| OK | Las fechas coinciden en todos los documentos |
| WARNING | Diferencia menor (dias) entre documentos |
| ERROR | Diferencia significativa (meses/anos) o el perito puso otra fecha |

### 1.2 Mecanica del accidente

**Que verificar:** La mecanica del accidente debe coincidir entre demanda y pericia.

| Resultado | Condicion |
|-----------|-----------|
| OK | La mecanica es consistente |
| WARNING | El perito resumio la mecanica diferente pero no cambia la esencia |
| ERROR | La mecanica en la pericia contradice la demanda |

### 1.3 Zona lesionada (lateralidad)

**CRITICO** — Este es uno de los errores mas graves.

**Que verificar:** DERECHA/IZQUIERDA debe coincidir entre demanda y pericia.

| Resultado | Condicion |
|-----------|-----------|
| OK | La zona y lateralidad coinciden |
| ERROR | El perito evaluo la zona CONTRARIA (ej: demanda dice rodilla derecha, perito examino izquierda) |
| ERROR | El perito no especifico lateralidad cuando la demanda si lo hace |

**Como verificar:**
1. Buscar en la demanda: "rodilla DERECHA", "hombro IZQUIERDO", etc.
2. Buscar en la pericia: la zona evaluada con su lateralidad
3. Comparar. Si no coinciden → ERROR GRAVE

### 1.4 Alta medica

**Que verificar:** Si la pericia menciona fecha de alta medica.

| Resultado | Condicion |
|-----------|-----------|
| OK | La pericia menciona la fecha de alta |
| WARNING | La pericia no menciona alta medica |
| ERROR | La fecha de alta difiere significativamente de la demanda/SRT |

## SECCION 2: Incapacidad Fisica

### 2.1 Patologias constatadas vs reclamadas

**Que verificar:** Cruzar las patologias reclamadas en la demanda con las constatadas por el perito.

| Resultado | Condicion |
|-----------|-----------|
| OK | El perito constato las mismas patologias reclamadas |
| WARNING | El perito no constato alguna patologia reclamada (pero tiene fundamento) |
| ERROR | El perito no constato una patologia reclamada sin dar fundamento |
| ERROR | El perito constato patologias distintas a las reclamadas sin explicar |

### 2.2 Comparacion con SRT

**Que verificar:** Si hay dictamen SRT, comparar con la pericia.

| Resultado | Condicion |
|-----------|-----------|
| OK/FAVORABLE | La SRT no dio % pero el perito si → bueno para el actor |
| OK | Los porcentajes son similares |
| WARNING | Diferencia menor entre SRT y pericia |
| ERROR | La SRT dio incapacidad y el perito no → IMPUGNAR |
| ERROR | La pericia da menos que la SRT sin justificacion |

### 2.3 Limitacion funcional (grados de movilidad vs baremo)

**Que verificar:** Los grados de movilidad informados por el perito deben corresponder al porcentaje asignado segun el Decreto 659/96.

**Como verificar:**
1. Identificar cada articulacion evaluada
2. Anotar los grados de movilidad informados (ej: "flexion 0-120°")
3. Consultar la tabla del baremo para esa articulacion
4. Verificar si el % asignado corresponde a los grados informados

| Resultado | Condicion |
|-----------|-----------|
| OK | Los grados informados corresponden al % asignado segun baremo |
| ERROR (a favor) | Los grados ameritan MAS % del que asigno el perito → pedir mas |
| ERROR (en contra) | Los grados NO ameritan el % asignado → puede ser cuestionado por la ART |
| WARNING | El perito no informo grados exactos, solo puso el % |

**Ejemplo:**
- Perito informa flexion de rodilla 0-120° y asigna 2% → ERROR: segun baremo es 5%
- Perito informa flexion de rodilla 0-140° y asigna 5% → ERROR: segun baremo es 2%

### 2.4 Acumulacion patologia + limitacion funcional

**Que verificar:** Verificar si el Decreto permite acumular patologia y limitacion funcional.

| Resultado | Condicion |
|-----------|-----------|
| OK | El perito sumo correctamente o no aplica |
| WARNING | El perito sumo patologia + limitacion cuando el baremo ya la incluye |
| ERROR | Podria haberse sumado y no se hizo |

**REGLA:** La inestabilidad de rodilla YA INCLUYE la limitacion funcional. No se acumulan.

### 2.5 Control especifico de rodilla

**Solo aplica si la lesion es en rodilla.**

**Que verificar** (checklist):

| Item | Verificar | Si positivo |
|------|-----------|-------------|
| Liquido intraarticular en estudios | RMN, ecografia | Reclamar HIDRARTROSIS (5-8%) |
| Sinovitis en estudios | RMN, ecografia | Reclamar SINOVITIS CRONICA (5-8%) |
| Lesion meniscal con signos objetivos | McMurray+, Apley+, hipotrofia, bloqueo | Reclamar SINDROME MENISCAL (8-10%) |
| Operaron de meniscos | Historia clinica, cirugia | Reclamar MENISCECTOMIA sin secuelas (3-6%) o con secuelas (10-15%) |
| Maniobras de inestabilidad | Lachman, cajon anterior/posterior, bostezo | Reclamar inestabilidad (5-30%) |

**Signos OBJETIVOS** (dan incapacidad sin necesidad de limitacion funcional):
- Hidrartrosis / derrame
- Atrofia del cuadriceps (medir perimetro muslo)
- Bloqueo articular
- Maniobras meniscales positivas
- Inestabilidad ligamentaria

**Signos SUBJETIVOS** (no dan incapacidad por si solos):
- Dolor referido
- Crepitacion sola

### 2.6 Control de inestabilidad articular

**Aplica a:** hombro, codo, muneca, rodilla, tobillo, cadera.

| Resultado | Condicion |
|-----------|-----------|
| OK | El perito evaluo inestabilidad y las maniobras fueron negativas |
| OK | El perito constato inestabilidad y la incluyo |
| WARNING | El perito no evaluo inestabilidad pero no hay indicios |
| ERROR | Hay indicios de inestabilidad (estudios, clinica) y el perito no la evaluo |
| ERROR | Las maniobras fueron positivas pero el perito no dio incapacidad por inestabilidad |

### 2.7 Control de hernias

| Resultado | Condicion |
|-----------|-----------|
| OK | No se reclamaron hernias, o el perito las evaluo |
| ERROR | Se reclamaron hernias en la demanda y el perito no las evaluo |
| ERROR | Los estudios muestran hernias y el perito las ignoro |

### 2.8 Control de cicatrices

| Resultado | Condicion |
|-----------|-----------|
| OK | No se reclamaron cicatrices, o el perito las evaluo |
| ERROR | Se reclamaron cicatrices en la demanda y el perito no las evaluo |

### 2.9 Control de lesion de nervios perifericos / EMG

| Resultado | Condicion |
|-----------|-----------|
| OK | No se reclamo lesion de nervios, o el perito la evaluo |
| WARNING | El perito no solicito EMG pero no hay indicios clinicos |
| ERROR | Se reclamo lesion de nervios y el perito no la evaluo |
| ERROR | Hay parestesias/hormigueos y el perito no solicito EMG |
| ERROR | El EMG muestra alteraciones y el perito no dio incapacidad |

### 2.10 Control de miembro habil

**Solo aplica a miembros SUPERIORES (brazo, antebrazo, mano, muneca, codo, hombro).**

**Que verificar:** Si el miembro lesionado es el HABIL (derecho en diestros, izquierdo en zurdos), el perito debe sumar 5%.

| Resultado | Condicion |
|-----------|-----------|
| OK | Sumo el 5% por miembro habil |
| OK | No aplica (miembro inferior, o no es el habil) |
| ERROR | Es el miembro habil y NO sumo el 5% |

## SECCION 3: Incapacidad Psiquica

### 3.1 Evaluacion de incapacidad psiquica

| Resultado | Condicion |
|-----------|-----------|
| OK | Hay pericia psicologica separada que evaluo |
| OK | El perito medico evaluo incapacidad psiquica |
| WARNING | Se reclamo incapacidad psiquica y nadie la evaluo |
| ERROR | Se reclamo y se nego sin fundamento |

### 3.2 Grado de reaccion vivencial

**Verificar que el % corresponda al grado:**

| Grado | Porcentaje correcto |
|-------|-------------------|
| Grado I | 0% |
| Grado II | 10% |
| Grado II-III | 15% |
| Grado III | 20% |
| Grado III-IV | 25% |
| Grado IV | 30% |

| Resultado | Condicion |
|-----------|-----------|
| OK | El % corresponde al grado informado |
| ERROR | El % no corresponde al grado (ej: Grado III pero puso 10%) |

### 3.3 Alteraciones cognitivas

**REGLA CLAVE:** Si hay alteraciones de memoria Y/O concentracion/atencion → NO puede ser Grado II.

| Resultado | Condicion |
|-----------|-----------|
| OK | No hay alteraciones cognitivas, o el grado es III+ |
| ERROR | Hay alteraciones de memoria/concentracion y el perito puso Grado II (10%) |
| ERROR | Deberia ser al menos Grado III (20%) |

## SECCION 4: Causalidad y Permanencia

### 4.1 Relacion de causalidad fisica

**Que verificar:** El perito DEBE vincular las lesiones fisicas con el accidente.

**Frases a buscar:** "producto del accidente", "relacionado causalmente", "guarda relacion causal", "tiene nexo causal", "vinculado con el hecho denunciado"

| Resultado | Condicion |
|-----------|-----------|
| OK | El perito establecio causalidad claramente |
| WARNING | El perito uso lenguaje ambiguo ("podria estar relacionado") |
| ERROR | El perito NO establecio causalidad |
| ERROR | El perito NEGO causalidad sin fundamento adecuado |

### 4.2 Relacion de causalidad psiquica

Mismo criterio que 4.1 pero para el dano psiquico.

| Resultado | Condicion |
|-----------|-----------|
| OK | Establecio causalidad psiquica |
| WARNING | No evaluo causalidad psiquica (pero tampoco evaluo lo psiquico) |
| ERROR | Evaluo incapacidad psiquica pero NO establecio causalidad |

### 4.3 Incapacidad permanente

**Que verificar:** El perito DEBE decir que la incapacidad es PERMANENTE o IRREVERSIBLE.

**Frases a buscar:** "permanente", "irreversible", "definitiva", "no susceptible de mejoria"

| Resultado | Condicion |
|-----------|-----------|
| OK | Dijo que es permanente/irreversible |
| WARNING | No lo menciono expresamente |
| ERROR | Dijo que es transitoria o susceptible de mejoria (salvo que sea correcto) |

### 4.4 Causalidad negada

Si el perito NEGO causalidad para alguna patologia:

| Resultado | Condicion |
|-----------|-----------|
| OK | La negacion tiene fundamento cientifico adecuado |
| ERROR | La negacion no tiene fundamento o contradice los estudios |
| ERROR | Nego causalidad para la patologia principal reclamada |

## SECCION 5: Baremo

### 5.1 Baremo utilizado

| Resultado | Condicion |
|-----------|-----------|
| OK | Uso Decreto 659/96 |
| OK | No menciono cual uso (se presume 659/96) |
| ERROR | Uso otro baremo (AMA, Altube, Rubinstein, etc.) |
| ERROR | Uso Decreto 549/2025 cuando el accidente es anterior a su entrada en vigencia |

### 5.2 Decreto 549/2025 aplicado retroactivamente

**CRITICO** — Este control se activa cuando el perito utilizo el Decreto 549/2025 en un caso donde el accidente ocurrio antes de su entrada en vigencia. Es uno de los errores mas impactantes porque afecta TODA la determinacion de incapacidad.

**Como detectar:**
1. Buscar en la pericia menciones a "Decreto 549/2025", "Decreto 549", "549/25", "nueva tabla", "tabla vigente"
2. Verificar la fecha del accidente/primera manifestacion invalidante
3. Si el accidente es anterior a la entrada en vigencia del Decreto 549/2025 → ERROR

| Resultado | Condicion |
|-----------|-----------|
| OK | El accidente es posterior al 549/2025, o el perito uso 659/96 |
| ERROR | El accidente es ANTERIOR al 549/2025 y el perito aplico este baremo |

**Consecuencias del error:**
- Los porcentajes del 549/2025 son sustancialmente inferiores al 659/96 para las mismas limitaciones
- El 549/2025 impone la formula de capacidad restante (Balthazar) de manera GENERALIZADA a todas las incapacidades parciales, incluso las del mismo siniestro
- El 659/96 SOLO aplica Balthazar para incapacidades de siniestros DIFERENTES y PREVIOS
- La combinacion de ambos efectos puede reducir drasticamente la incapacidad total
- Puede hacer caer al actor por debajo de umbrales legales criticos (50% T.O., 66% T.O.)

**REGLA**: El baremo aplicable es el vigente al momento de la PRIMERA MANIFESTACION INVALIDANTE (fecha del accidente o toma de conocimiento de la enfermedad profesional), NO el vigente al momento de la pericia. Principio de irretroactividad (art. 7 CCyCN).

### 5.3 Impacto en umbrales legales

**Solo aplica si se detecto ERROR en 5.2.**

Recalcular la incapacidad total usando 659/96 con suma aritmetica y verificar si supera umbrales criticos:

| Umbral | Consecuencia legal | Verificar |
|--------|-------------------|-----------|
| 50% T.O. | Art. 14 inc. 2 a) Ley 24.557: prestacion de pago mensual complementaria | Si la incapacidad con 659/96 supera 50% pero con 549/2025 queda debajo |
| 66% T.O. | Gran invalidez, prestacion especial | Si la incapacidad con 659/96 supera 66% pero con 549/2025 queda debajo |

| Resultado | Condicion |
|-----------|-----------|
| OK | No hay cambio de umbral |
| ERROR GRAVE | La correccion del baremo hace superar un umbral legal critico |

**Si se detecta cambio de umbral**: Enfatizar en la observacion que la aplicacion erronea del 549/2025 no solo perjudica cuantitativamente sino que tiene consecuencias CUALITATIVAS determinantes.

## SECCION 6: Factores de Ponderacion

**Esta es la seccion con MAS ERRORES de los peritos.**

### 6.1 Aplicacion de factores

| Resultado | Condicion |
|-----------|-----------|
| OK | Aplico los tres factores (edad, dificultad, recalificacion) |
| ERROR | No aplico ningun factor de ponderacion |
| ERROR | Solo aplico algunos factores |

### 6.2 Factor edad

**REGLA:** El factor edad es una TABLA con rangos por franja etaria:

| Franja etaria | Rango |
|---------------|-------|
| Menos de 21 anos | 0-4% |
| De 21 a 30 anos | 0-3% |
| De 31 y mas anos | 0-2% |

Se verifica: (a) que el % este dentro del rango de la tabla para la edad del actor, (b) que se haya sumado de forma ARITMETICA DIRECTA (como puntos porcentuales que se adicionan al total), NO como porcentaje de la incapacidad.

| Resultado | Condicion |
|-----------|-----------|
| OK | El % esta en el rango correcto y se sumo de forma aritmetica directa |
| ERROR | El % esta fuera del rango de la tabla (ej: 3% para 31+ cuando el max es 2%) |
| ERROR | Aplico factor edad como PORCENTAJE de la incapacidad (ej: 2% de 30% = 0.6%) |
| ERROR | No aplico factor edad |

**Ejemplo de error tipico:**
- Actor de 45 anos (31+), incapacidad 30%
- CORRECTO: Factor edad = hasta 2% (sumado directo como puntos porcentuales)
- INCORRECTO: Factor edad = 3% (fuera del rango 0-2%)
- INCORRECTO: Factor edad = 2% de 30% = 0.6% (aplicacion porcentual)

### 6.3 Factor dificultad para tareas habituales

| Grado | Porcentaje sobre incapacidad |
|-------|------------------------------|
| Leve | Hasta 10% |
| Intermedia | 11-15% |
| Alta | 16-20% |

| Resultado | Condicion |
|-----------|-----------|
| OK | El porcentaje corresponde al grado indicado |
| ERROR | Puso "intermedia" pero dio 10% (deberia ser 11-15%) |
| ERROR | Puso "alta" pero dio menos de 16% |
| ERROR | No aplico factor dificultad |

### 6.4 Factores sobre incapacidad psiquica

**REGLA CRITICA:** Los factores dificultad y recalificacion TAMBIEN aplican sobre la incapacidad psiquica. No solo sobre la fisica.

| Resultado | Condicion |
|-----------|-----------|
| OK | Aplico factores sobre fisica Y psiquica |
| ERROR | Solo aplico factores sobre la fisica |
| N/A | No hay incapacidad psiquica |

## SECCION 7: Calculo

### 7.1 Metodo de capacidad restante (Balthazar)

**Bajo el Decreto 659/96, solo se aplica si:**
- Hubo siniestro anterior con incapacidad mensurada
- Examen preocupacional constato lesiones
- Es "gran siniestrado" (>66%)

**ATENCION: Decreto 549/2025 vs 659/96**
El Decreto 549/2025 impone la formula de capacidad restante de manera GENERALIZADA a TODAS las incapacidades parciales, incluso las derivadas del MISMO siniestro. El Decreto 659/96 la reserva EXCLUSIVAMENTE para combinar incapacidades de siniestros DIFERENTES y PREVIOS. Si el perito aplico capacidad restante porque uso el 549/2025, este error es CONSECUENCIA del error de baremo (control 5.2) y debe impugnarse conjuntamente.

| Resultado | Condicion |
|-----------|-----------|
| OK | Lo aplico correctamente (habia incapacidad previa documentada) |
| OK | No lo aplico (no habia incapacidad previa) |
| ERROR | Lo aplico SIN que haya incapacidad previa → baja el total injustamente |
| ERROR | Lo aplico a incapacidades del MISMO siniestro porque uso metodologia del 549/2025 |
| ERROR | No lo aplico cuando si habia incapacidad previa |

**Como cuantificar el perjuicio:** Comparar suma aritmetica vs capacidad restante. Ejemplo tipico:
- Incapacidades parciales: 12% + 9% + 6% = 27% (suma aritmetica bajo 659/96)
- Con capacidad restante: 12% + 7.92% + 3.84% = 23.76% (reduccion de 3+ puntos)
- La diferencia se amplifica cuantas mas incapacidades parciales haya

### 7.2 Suma aritmetica

**Verificar:** Incapacidad total = Fisica + Psiquica + Factor Edad + Factor Dificultad + Factor Recalificacion

| Resultado | Condicion |
|-----------|-----------|
| OK | La suma es correcta |
| ERROR | La suma tiene errores aritmeticos |
| ERROR | No incluyo todos los componentes en la suma |

## SECCION 8: Incapacidad Integral (Preexistencias)

**Solo aplica si hay preexistencias.**

Evaluar si corresponde reclamar incapacidad integral conforme Art. 45 inc c Ley 24.557: la ART de la ultima contingencia debe pagar segun incapacidad integral si cambia la modalidad de pago.

| Resultado | Condicion |
|-----------|-----------|
| OK | No hay preexistencias |
| WARNING | Hay preexistencias, evaluar si conviene reclamar integral |
| ERROR | Hay preexistencias y el perito no las considero |

---

## Decision Final

Despues de completar TODOS los controles:

### HAY QUE IMPUGNAR si:
- Hay 1 o mas items con resultado ERROR
- Y la pericia NO es altamente favorable

### NO CONVIENE IMPUGNAR si:
- La pericia da alta incapacidad (>30%) con buena causalidad y permanencia
- Los errores son menores y corregirlos no aumentaria significativamente el monto
- Impugnar podria resultar en una pericia PEOR

### IMPUGNAR CON CAUTELA si:
- La pericia es parcialmente favorable pero tiene errores en items puntuales
- Solo impugnar los puntos especificos, no la pericia en general
